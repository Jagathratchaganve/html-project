# Starbucks HTML Mini Project

Welcome to the **Starbucks Static Website Project**.  
This project is a clean, beginner‑friendly, multi‑page HTML website styled using CSS.  
It showcases proper semantic HTML structure, tables, forms, navigation, time elements, and a fully styled layout wrapped inside a document-style container.

---

## 📌 Features

### ✅ Multi‑Page Website
- `Home`  
- `Menu`  
- `Store Hours`  
- `Contact Us`

### ✅ Fully Semantic HTML5
- `<section>`, `<article>`, `<header>`, `<footer>`, `<nav>`, `<figure>`, `<address>`
- `<time datetime="">` used properly

### ✅ Styled Tables
- Menu table  
- Store hours table  
- Borders around caption  
- Clean readable monospace table design

### ✅ Contact Form
- Form with name, email, and message fields  
- Uses `<fieldset>` and `<legend>`  
- Submits to `https://httpbin.org/post` for testing  
- Reset & submit buttons included  

### ✅ Document‑Style Layout (Page Container)
A centered page box resembling a document using CSS:
- Outer border  
- Rounded corners  
- Max‑width for readable layout  
- Light shadow  
- Clean background  

### ✅ Navigation Links
Each page links to:
- Home  
- About  
- Menu  
- Store Hours  
- Contact page  

### ✅ Clean CSS
- Modern, readable design  
- Green Starbucks‑style theme  
- Hover effects  
- Styled captions, forms, tables  
- Centralized figure images
